# 🇩🇪 My German Portfolio

A high-performance software engineering portfolio tailored for opportunities in the German tech market.

Built with a modern frontend architecture, advanced UI systems, interactive 3D visualization, and Swiss design principles.

---

## 👨‍💻 About This Project

This portfolio demonstrates:

* Scalable frontend architecture
* Strong TypeScript-based development
* Performance-conscious UI engineering
* Interactive 3D visualization
* Clean, maintainable project structure

It is designed to reflect production-level standards expected in modern European engineering teams.

---

## 🏗️ Core Framework & Language

* **React 18** — Component-driven UI architecture
* **TypeScript** — End-to-end type safety
* **Vite** — Fast development server & optimized builds

---

## 🎨 Styling & Design System

* **Tailwind CSS** — Utility-first styling
* **Lucide React** — Consistent SVG icon system
* **Google Fonts**

  * Inter (UI typography)
  * JetBrains Mono (technical data)

---

## 🚀 Animations & 3D Graphics

* **Framer Motion** — Smooth transitions and micro-interactions
* **Three.js + React Three Fiber** — Interactive 3D Skill Universe
* **React Three Drei** — 3D helper abstractions

---

## 📊 Data Visualization & Integrations

* **Recharts** — Technical Radar Chart
* **LeetCard API** — Live LeetCode integration
* **GitHub Charts** — Dynamic contribution heatmaps
* **Web SpeechSynthesis API** — German pronunciation feature

---

## 📐 Design Philosophy

The portfolio follows Swiss Design principles:

* Grid-based layout precision
* Typography-driven hierarchy
* Minimalist monochrome palette
* Single customizable accent color
* Functional clarity over decoration

---

## ⚙️ Project Structure

```
src/
 ├── components/
 ├── pages/
 ├── assets/
 ├── App.tsx
 └── main.tsx
```

---

## 🛠️ Local Development

Install dependencies:

```
npm install
```

Start development server:

```
npm run dev
```

Build for production:

```
npm run build
```
